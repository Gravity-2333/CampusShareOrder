# CampusShareOrder（校园拼单平台）

CampusShareOrder 是一个面向校园场景的拼单平台，支持学生发起拼单、参与拼单、伪支付、伪退款、凭证上传、收货确认、投诉处理以及信用分展示等功能。

本项目为软件工程课程项目，重点实现一个“完整业务闭环 + 简化实现”的拼单系统，适配两周开发周期。

------

## 一、项目背景

在校园生活中，学生经常通过微信群、QQ群进行拼单（外卖、团购等）。
但这种方式存在以下问题：

-   拼单信息分散，难以查找
-   拼单状态不清晰，容易错过
-   转账混乱，容易产生纠纷
-   缺乏信用机制，容易“放鸽子”
-   出现问题无法有效处理

本项目通过构建一个专门的拼单平台，统一管理拼单流程，提高效率并减少纠纷。

------

## 二、核心功能

### 用户模块

-   注册 / 登录
-   学号实名认证
-   个人信息管理
-   信用分展示

### 拼单模块

-   发起拼单
-   浏览拼单列表
-   查看拼单详情
-   加入拼单 / 退出拼单（未成团）
-   自动成团
-   超时自动取消
-   上传订单凭证
-   确认送达
-   确认收货（支持自动确认）
-   拼单完成

### 信任与投诉模块

-   信用分计算与展示
-   提交投诉
-   投诉处理
-   投诉结果通知

### 管理模块

-   管理员登录
-   用户管理（封禁 / 解封）
-   拼单管理
-   投诉处理
-   支付 / 退款记录查看
-   操作日志查看

------

## 三、核心流程

正常流程：
创建拼单 → 加入拼单 → 支付 → 成团 → 上传凭证 → 确认送达 → 确认收货 → 完成

异常流程：
未成团 → 自动取消 → 自动退款
发起人异常 → 投诉 → 管理员处理 → 退款 + 扣分
